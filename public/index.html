Help

Asking
  • One prompt goes to Claude, Gemini, and ChatGPT via their CLIs (your subscriptions, no API billing).
  • Enter sends, Shift+Enter adds a newline.
  • @Claude / @Gemini / @Codex (or @ChatGPT) sends to only those models. No mention = your default models (Settings).
  • Drag files anywhere (or click the paperclip) to attach text, code, images, or PDFs.
Two modes (pick per chat)
  • + New chat starts a Standard chat: the classic three side-by-side panes, full answers, one row per prompt.
  • + New tree chat starts a Tree of thoughts chat: a branching node graph (below).
  • A chat keeps the mode it was created in; you can't switch a single chat between modes.
Standard mode
  • Each prompt's three answers show full, side by side; ask again to continue.
  • ✓ on an answer makes the next prompt build on that one answer (all models continue from it); click again to clear.
  • Per answer: copy, ! criticize (others critique it), Humanize, × delete. × on the prompt deletes the whole turn.
Tree of thoughts mode
  • The chat is drawn as a tree: lines connect each answer to the answer it grew from, so you see exactly which response each branch came from.
  • Each answer is a compact card (tinted) showing a one-sentence summary the model wrote.
  • Click an answer to open it (full text, white) and select it; click again to close it. With an answer selected, the next prompt makes all models build on that one answer (they converge into three new cards under it).
  • Click a prompt node to select it (and open all its answers). The next prompt then has each model continue its own answer in that group, giving three parallel lines, parent to child.
  • With nothing selected, a new prompt continues from the last answers the same way (three parallel lines, each model from its own).
  • Other answers stay. Select any answer or prompt later and ask again to explore a different branch.
  • A node or prompt that already has follow-ups can't be branched again. Delete its follow-ups first (× on the follow-up prompt) to ask a different question there.
  • ⤢ on a card opens/closes it without selecting. Expand all / Collapse all (top right) open or close every card. New answers open automatically while they stream.
Prompt node (hover)
  • Compare: semantic synthesis of that prompt's answers (consensus, differences, unique points), shown in a panel.
  • Criticize: every model critiques the others; the per-card ! (in the expanded view) has the others critique just that answer.
  • ×: deletes that prompt, its answers, and everything branched from them.
  • Synthesis and critiques are saved with the chat.
Sidebar
  • New chat (standard) / New tree chat start a fresh chat in that mode; Temporary chat is never saved.
  • Hover a chat: pencil renames (or double-click the title), × deletes.
  • Skills: enable/disable each CLI's skills (ⓘ shows descriptions).
  • Settings: appearance (font, size, night mode), humanize-all toggle, default models, base model per service, and personalization (standing instructions sent with every prompt).
